# z2scanner
z2 scanner

## Requirement
- python3.7.4
- venv, pipenv

## Prepare for development
#### Enter devel environment with venv
`source devel/bin/activate`
#### Escape devel environment
`deactivate`

#### Enter pipenv
`pipenv shell`

#### Install pip package under under pipenv
`pipenv install PACKAGE_NAME`

## Run Test
python test/test01.py
